## 🖼️ Preview

> A modern, responsive personal portfolio built with **React + Vite + Tailwind CSS** featuring smooth animations, a chatbot assistant, splash screen, and more.

---

## 🚀 Features

- 🎬 **Splash Screen** — Animated intro on load
- 🧭 **Sticky Navbar** — Smooth scroll navigation with active state
- 🦸 **Hero Section** — Typewriter role animation + profile avatar
- 👤 **About** — Personal intro with floating badges
- 🛠️ **Skills** — Tech stack with devicon icons grouped by category
- 🗂️ **Projects** — Cards with modal popups (CitySolve, Gallery Hub, Calculator Pro)
- 💼 **Experience** — Internship @ CodeAlpha + 🏆 Hackathon Winner
- 🎓 **Education** — Timeline layout with institution logos
- 📄 **Resume** — Download & open PDF buttons
- 📬 **Contact** — Mailto form + social links
- 🤖 **AI Chatbot** — Smart virtual assistant that answers questions about Yash
- ⬆️ **Back to Top** — Floating scroll-to-top button

## ⚙️ Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/yashsoni972/Portfolio.git

# 2. Navigate into the folder
cd Portfolio

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev
```

Then open **http://localhost:5173** in your browser.

---

## 🏗️ Build for Production

```bash
npm run build
```
